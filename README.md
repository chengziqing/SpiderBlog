SpiderBlog
==========
最近喜欢看博客，博客这东西巩俐相对较小，一般博主都是出于分享和真心总结的新的体会，真实而有质量。找一个好的博客，对你应该帮助不少。
我找博客，都是从一个牛人的博客友情链接找，相信物以类聚，牛人博客的友情链接定也是牛人。从而我也确实找到不少好博客。而这也突发了我这个灵感，通过蜘蛛爬虫模拟我这种行为，把所以的优秀博客找出来。然后把这些博客通过网站展示给网友访问。这可以节省像我这样苦于找博客的人。
